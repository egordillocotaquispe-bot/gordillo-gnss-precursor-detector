# gordillo-gnss-precursor-detector
Algorithm to detect anomalous accelerations in GNSS time series using sliding windows (30 days) and the Mann-Kendall test. Validated using data from eight stations during the Arequipa (2001, Mw 8.4), Iquique (2014, Mw 8.2), and Arequipa (2018, Mw 7.1) earthquakes. A shuffle test confirms that the signal is not noise.
