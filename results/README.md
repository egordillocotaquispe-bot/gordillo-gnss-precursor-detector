# Results

This folder contains the output files generated by the validation scripts and tests.

The files are not included in this repository because they are produced locally when running the code. This folder is kept here to maintain the repository structure and to store your own results.

## Expected files

When you run the validation scripts, the following files may be generated:

| File name                      | Description |
|--------------------------------|-------------|
| `shuffle_test_resultado.png`   | Histogram of p-values from the shuffle test |
| `resultados.csv`               | Summary of window slopes and dates |
| `arequipa_2001_plot.png`       | Trend plot for the Arequipa 2001 validation |
| `iquique_2014_plot.png`        | Trend plot for the Iquique 2014 validation |
| `arequipa_2018_plot.png`       | Trend plot for the Arequipa 2018 validation |

## Usage

After running any validation or test script, copy or save the output files to this folder.

Example:

```bash
python validation/arequipa_2001.py
# The plot will be saved automatically to this folder if the script is configured to do so.
